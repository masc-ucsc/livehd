Policy carried forward (unchanged)

  • Asymmetric soundness: hard-error only on a definitive refutation that cannot be an artifact (stateless cone, or reachable-by-BMC witness, after assumes). Budget / Unknown / possibly-unreachable never error — a budget mistune can never turn a good build into a false error.
  • Only proven assumes are hypotheses (no circular self-proof); the consistency check runs before any removal; contradictory assume set ⇒ warning + removals suppressed. assume_nocheck_synth facts are never proof hypotheses — they are synthesis-only don't-cares by explicit user fiat; assume_nocheck_formal is ignored by this tier entirely.
