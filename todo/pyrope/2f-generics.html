2f-generics Generic <T> per-call-site specialization

Explicit generic parameters on lambdas: comb f<T>(a:T, b:T) -> (r). Untyped-parameter templates already specialize per call site (deferred templates minting foo__u8-style modules), but the per-T body substitution is pending (prp2lnast comment: "deferred follow-up goal").

Failing example (comptime)

comb triadd<T>(a:T, b:T, c:T) -> (r) { r = a + b + c }

cassert(triadd(1, 2, 3) == 6)         // T = int
cassert(triadd(u8(1), u8(2), u8(3)) == 6)  // T = u8
// triadd(1, "a", 3) — error: T must unify
