3f-temporal Temporal library (past/next/rose/fell/…)

past[n](x) (compiler inserts n flops), next[n](x), rose/fell/stable/changed/eventually[w]/always[w], and the attribute sugar sig.[rising]/.[falling]/.[changed]. Everything except past is debug-only. No semantic handling exists anywhere today (zero hits in prp2lnast/upass).

Failing example (pyrope/verilog pair)

// delay2.prp
mod delay2(a:u8) -> (out:u8@[2]) {
  out = past[2](a)
}
// delay2.v — should LEC-match
module delay2(input clk, input reset, input [7:0] a, output reg [7:0] out);
  reg [7:0] t1;
  always @(posedge clk) begin
    if (reset) begin t1 <= 8'b0; out <= 8'b0; end
    else       begin t1 <= a;    out <= t1;   end
  end
endmodule
