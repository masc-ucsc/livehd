3f-verif_extras covercase, in-language lec(), requires/ensures

assert/cassert/cover work; the rest of the verification vocabulary does not: covercase (grouped covers, one member must hold each cycle), in-language lec(a, b) combinational equivalence, and requires/ensures lambda pre/postconditions. Related: 4b (emit assert/assume/coverpoint into slop).

Failing example (comptime-provable)

comb fun1(a, b) -> (r) { r = a | b }
comb fun2(a, b) -> (r) { r = ~(~a | ~b) }
lec(fun1, fun2)              // De Morgan: provable at compile time

mut x:u8 = 3
covercase(NUM, (x & 1) == 1, "odd")
covercase(NUM, (x & 1) == 0, "even")   // group total: one true per cycle

comb div8(a:u8) -> (q:u8) {
  requires(a != 0)
  q = a >> 3
  ensures(q <= a)
}
